# 🔥 Do-Rering FRONT
> **하이미디어 아카데미 인공지능(AI-X) 프로젝트 부트캠프 3기** <br/> **개발기간: 2024.2 ~ 2024.3**

<br/>

## 🎤 소개
#### 동화 생성과 더빙을 통해 부모와 자녀를 이어주다!
도리링은 AI를 활용한 동화 생성과 인공지능 및 부모 목소리를 이용한 더빙 서비스를 제공함으로서<br/>
단순한 스토리텔링을 넘어, 부모와 자녀 간 감정적인 유대를 강화하는 매개체가 됩니다.
